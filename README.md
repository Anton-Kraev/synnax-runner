# Telegram Bot для планирования Python скриптов

Этот Telegram-бот позволяет пользователям загружать Python файлы и планировать их ежедневное выполнение в 6:00 UTC.

## Функции

- 📤 Загрузка Python файлов через Telegram
- ⏰ Планирование ежедневного выполнения в 6:00 UTC
- 🔄 Автоматическая остановка предыдущих задач при загрузке нового файла
- 📢 Уведомления о начале выполнения скрипта
- ✅ Отправка результатов выполнения (успех/ошибки)
- 🌐 Интеграция с внешним API (заглушка)

## Требования

- Go 1.21+
- Python 3
- Telegram Bot Token

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd synnax-runner
```

2. Установите зависимости:
```bash
go mod tidy
```

3. Создайте Telegram бота через @BotFather и получите токен

4. Запустите бота одним из способов:

### Через переменную окружения:
```bash
export TELEGRAM_BOT_TOKEN="your_bot_token_here"
go run main.go
```

### Через CLI флаг:
```bash
go run main.go -token "your_bot_token_here"
# или
go run main.go -t "your_bot_token_here"
```

### Через Makefile:
```bash
make run
```

## Использование

1. Отправьте `/start` боту для начала работы
2. Используйте команду `/schedule` для планирования нового скрипта
3. Настройте расписание выполнения (формат cron)
4. Установите таймаут выполнения скрипта
5. Отправьте Python файл (.py) боту
6. Бот автоматически запланирует выполнение согласно вашим настройкам

## Команды бота

- `/start` - Начало работы с ботом
- `/schedule` - Планирование нового Python скрипта
- `/help` - Показать справку по использованию

## Особенности

- **Один пользователь**: Бот работает только с первым пользователем, который отправил сообщение
- **Автоматическая блокировка**: Другие пользователи получают сообщение о недоступности
- При загрузке нового файла предыдущий автоматически останавливается
- Настраиваемое расписание выполнения (формат cron)
- Настраиваемый таймаут выполнения (от 30 секунд до 1 часа)
- Все уведомления отправляются в UTC времени
- Результаты выполнения включают вывод скрипта и ошибки (если есть)
- Поддержка CLI флагов для настройки токена
- Современная библиотека telebot для работы с Telegram API

## Безопасность

- Файлы сохраняются в отдельных директориях для каждого пользователя
- Выполнение скриптов ограничено по времени
- Поддерживаются только Python файлы (.py)

## Разработка

Для добавления реального внешнего API замените функцию `callExternalAPI()` в `main.go` на вашу реализацию.
